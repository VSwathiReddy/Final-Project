Welcome to Our App

With this new Medicine Review web application, a user-friendly and high quality service will be provided to the users.

Get Started






About the medicine review web application

Our main motivation to develop this web app is to help users by providing medicine data so that they can be assured before taking any medication and also providing with other services related to medications.

Technologies used

Developing this app using frontend technologies using HTML, CSS, Bootstrap, JavaScript and backend technologies using Django framework, Python, PHP and MySQL database.

User Types

There will 3 types of users who can register to this web app and perform actions. Patient, Doctor and Medical Service Representative are the users.

Uses

This app will help users to verify the medication before taking it, store the prescrition along with other details and get notifications at the time to the time to take medicine to remind the user so that they will not forgot.

Services

Our application will help users by providing various services which are displayed here. Doctors will register and login to the application where they can view all request from patients and can generate prescription for each patient. This application will save prescription in user database as well as send to patient mail id. Medical Service Representative will signup and login to the application to add medicines details such as medicine name, formula, recommended dosage and side effects etc. Patients can view all this medicine details. Patients can register with the application and then login to send query to doctors and can view prescription from doctor. Patients can set reminders for medicines so that they receive notifications via email and can view those reminders anytime by login to application. Patients can view all medicines details by entering its names.

Medicine information

Medicine details such as medicine name, formula, recommended dosage, side effects, nearby medical store, etc. to crosscheck before taking medicine.

Pill Reminder

Can set reminders for medicines and can view those reminders in the app anytime and email notifications will be sent as a reminder.

Get Digital Prescription

A prescrition will be sent to email and view the prescription in the app anytime by sending a query to doctor.

Email Notifications

Prescription will be sent to patient through email and also everyday notifications will be sent as a reminder.

Send query and Chat functionality

Patients can view all the doctors available and talk to them to discuss about their situation.

Details stored in db

All the details such as user login details, prescription, medicine information, reminders are stored in the database.
